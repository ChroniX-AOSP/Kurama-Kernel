# MilkKernel - Action

**Dynamic Kernel, based on Vantom**

**For davinci | Mi 9T / Redmi K20**

## Install

- Download **[latest release](https://github.com/SchweGELBin/action_kernelsu_davinci/releases/latest/download/MilkKernel.zip)**
- Flash via Recovery

## Build

- Fork this repository
- Run Action "CI Build"
